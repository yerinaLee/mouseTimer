# mouseTimer

25~30초 사이의 랜덤시간 동안 마우스 또는 키보드에 작동이 없으면, 자동으로 마우스가 움직이고 간헐적으로 클릭이 되는 .exe 파일입니다.
파일 실행 후, 우측 하단의 프로그램목록에서 종료할 수 있습니다.
